[SIZE=6][B]Signature Once for XenForo 2.1.0+[/B][/SIZE]

[SIZE=5][B]Description[/B][/SIZE]
This add-on allows user signature to be shown only once per thread or once per page.

[SIZE=5][B]Requirements[/B][/SIZE]
[LIST]
[*]PHP 7.3.0+
[/LIST]

[SIZE=5][B]Options[/B][/SIZE]
[TABLE]
[TR]
[TH]Group[/TH]
[TH]Name[/TH]
[TH]Description[/TH]
[/TR]
[TR]
[TD]Threads, discussions and conversations[/TD]
[TD]Show user's signature once per conversation[/TD]
[TD]If unchecked, user's signature will be only be shown once per page.[/TD]
[/TR]
[TR]
[TD]Threads, discussions and conversations[/TD]
[TD]Show user's signature once per thread[/TD]
[TD]If unchecked, user's signature will be only be shown once per page.[/TD]
[/TR]
[/TABLE]
