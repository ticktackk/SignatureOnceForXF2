[HEADING=1]Signature Once for XenForo 2.2.0+[/HEADING]
[HEADING=1]Description[/HEADING]
This add-on allows user signature to be shown only once per thread or once per page.
[HEADING=1]Requirements[/HEADING]
[LIST]
[*]PHP 7.4.0+
[/LIST]
[HEADING=1]Options[/HEADING]
[HEADING=3]Threads, discussions and conversations[/HEADING]
[TABLE width="100%"]
[TR]
[th]Name[/th][th]Description[/th]
[/TR]
[TR]
[td]Show user's signature once per conversation[/td][td]If unchecked, user's signature will be only be shown once per page.[/td]
[/TR]
[TR]
[td]Show user's signature once per thread[/td][td]If unchecked, user's signature will be only be shown once per page.[/td]
[/TR]
[/TABLE]
[HEADING=1]Permissions[/HEADING]
[HEADING=3]Forum permissions[/HEADING]
[LIST]
[*]Bypass signature once
[/LIST]
[HEADING=3]Conversation permissions[/HEADING]
[LIST]
[*]Bypass signature once
[/LIST]
[HEADING=1]CLI Commands[/HEADING]
[TABLE width="100%"]
[TR]
[th]Command[/th][th]Description[/th]
[/TR]
[TR]
[td][ICODE]xf-rebuild:tck-signature-once-conversation-first-user-message-records[/ICODE][/td][td]Rebuilds conversation first user message records..[/td]
[/TR]
[TR]
[td][ICODE]xf-rebuild:tck-signature-once-thread-first-user-post-records[/ICODE][/td][td]Rebuilds thread first user post records.[/td]
[/TR]
[/TABLE]
